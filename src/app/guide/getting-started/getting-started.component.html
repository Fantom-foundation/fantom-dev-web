Getting started developing on Fantom

Fantom foundation is made up of a number of repositories. This will guide you through getting your development environment setup.

Though this guide is for the Ubuntu linux; it should be easy to retrofit to others.

Though this guide has been tested on macOS High Sierra; it will probably work on older versions.

Follow this guide on Windows 7+.

All grey background assumes you are running in an active Command Prompt (cmd.exe) Terminal session

Step 0: Install build dependencies 


sudo apt update
sudo apt install build-essential git-core tcl jq
Latest XCode and Command Line Tools 

/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
brew update
brew install git jq
Ensure you check "Add to PATH" whenever asked:
  1. Build Tools for Visual Studio 2017
  2. git

Test that it installed correctly with:


  git --version
  cl /?

Step 1: Install Go 


curl https://dl.google.com/go/go1.11.3.linux-amd64.tar.gz -o go.linux-amd64.tar.gz
tar -C /usr/local -xzf go.linux-amd64.tar.gz
sudo bash -c "GOPATH=$HOME/go >> /etc/environment"
      

brew install go

Official Go installer: go1.11.3.windows-amd64.msi

Step 2: Install Rust 


curl https://sh.rustup.rs -sSf | sh -s -- --default-toolchain nightly

Official Rust installer: https://win.rustup.rs/x86_64 


rustup set default-host x86_64-pc-windows-msvc
rustup install nightly
rustup default nightly

Step 3: Go Lachesis 


go get -v github.com/SamuelMarks/batch-ethkey


d="$GOPATH/src/github.com/Fantom-foundation"
mkdir -p "$d"
git clone https://github.com/Fantom-foundation/go-lachesis "$d/go-lachesis"
glide i
      

set d="%GOPATH%/src/github.com/Fantom-foundation"
md "%d%"
git clone https://github.com/Fantom-foundation/go-lachesis "%d%/go-lachesis"
glide i

Now you have everything you need to orchestrate with fantom-dev.
